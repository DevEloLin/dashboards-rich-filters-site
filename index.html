View on Marketplace Documentation

Features

Dashboard modules designed for project visibility

Project Health Dashboard

View overall project health metrics including completion rates, overdue issues, blocked items, and unassigned tasks in a consolidated dashboard.

Recent Activity

Track recent issue updates, status changes, and team activity to stay informed about project progress.

Issue Trends

Visualize issue creation and resolution trends over time to identify patterns and potential bottlenecks.

Sprint Progress

Monitor active sprint progress with burndown visualization, story point tracking, and completion status.

Statistics by Components

Break down issue statistics by component to understand workload distribution across different areas of your project.

Project Milestones

Track key project milestones and deliverables with timeline visualization and progress indicators.

Project Risk Analysis (Ishikawa)

Identify potential project risks using fishbone (Ishikawa) diagram visualization. Note: Risk analysis is provided as a reference tool and should be validated by project teams.

Multi-Select Filters

Filter dashboard data by projects, assignees, priorities, and statuses. Filters generate JQL queries automatically.
