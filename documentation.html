Complete guide to using Dashboards Rich Filters

Getting Started

Dashboards Rich Filters is a premium dashboard gadget for Atlassian Jira Cloud that transforms your project data into beautiful, executive-ready visualizations.

Quick Start: Install from Atlassian Marketplace, add the gadget to any dashboard, and start filtering your Jira data instantly.

Installation

From Atlassian Marketplace

  1. Go to Atlassian Marketplace
  2. Click "Get it now" or "Try it free"
  3. Select your Jira Cloud site
  4. Review and accept the permissions
  5. Click "Install"

Adding to Dashboard

  1. Navigate to any Jira Dashboard
  2. Click "Add gadget" in the top right
  3. Search for "Dashboards Rich Filters"
  4. Click "Add" to add it to your dashboard

Themes & Demo Mode

Dark Theme

Click the moon/sun icon in the gadget header to toggle between light and dark themes.

Demo Mode

Demo Mode masks all sensitive data with placeholder values - perfect for presentations, board meetings, and public demos.

Tip: Project names appear as "Project A", "Project B" in demo mode to protect sensitive information.
